Online pools

An online web application where users can create, share, and manage polls.

- Users are able to register an account;
- Users are able to login/logout from the system;
- Registered users are able to create a poll;
- Registered users are able to manage their polls (change Poll's name if anyone has not voted on it);
- Polls contain one image and several possible answers;
- Users are able to list, search and answer to polls;
- One user can't answer twice to the same poll;
- Poll owners and users that already answered a poll, are able to see the poll results;
- Poll owners are able to decide if the poll is public or private;
- Private polls don't appear in listings or searches (Only the poll URL should needed to access it);
- Polls can have more than one question;
- Possibility to share a poll using email.

Used technologies: PHP, HTML, JS, CSS, SQL, SQLite.

Execution date: 2014
